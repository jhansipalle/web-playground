## Installed VS Code Extensions (Session 1)

**Prettier - Code formatter** (`esbenp.prettier-vscode`)  
  Purpose: Auto-formats code on save to a consistent style. 
  Configured as the default formatter (see `.vscode/settings.json`).

**Live Server** (`ritwickdey.liveserver`)  
  Purpose: Serves your static site locally and reloads browser on file changes.

**ESLint** (`dbaeumer.vscode-eslint`)  
  Purpose: Lints JavaScript/TypeScript code and shows errors/warnings inline.

 ## Commit Convention

We follow [Conventional Commits](https://www.conventionalcommits.org/en/v1.0.0/)

Examples:
- `feat: add new feature`
- `fix: resolve a bug`
- `chore: update dependencies`
- `docs: update documentation`
